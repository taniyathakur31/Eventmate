# EventMate Frontend (React + Tailwind + Vite)

A fast, modern UI for **EventMate – Smart College Event Manager**.

## ✨ Features in this starter
- React (Vite) + Tailwind CSS
- Event creation (demo) using `localStorage`
- Registration page that generates a **QR** using `qrcode`
- Admin **Scanner** page using `react-qr-scanner`
- Pretty, responsive UI components

> ⚠️ This starter mocks data with `localStorage` so you can demo quickly.
> Replace storage with real APIs when you connect your Express + MongoDB backend.

## 🚀 Quick Start
```bash
# 1) Extract and install deps
npm install

# 2) Start dev server
npm run dev
```

## 🔗 Backend Endpoints (suggested)
- `POST /api/events` – create event
- `GET /api/events` – list events
- `POST /api/events/:id/register` – create registration, returns token
- `POST /api/events/:id/checkin` – verify token + mark attendance
- `GET /api/events/:id/stats` – aggregate counts (registered, checked-in)

## 🧩 Where to plug APIs in code?
- `src/pages/CreateEvent.jsx` → submit to `/api/events`
- `src/pages/Register.jsx` → call `/register` to get signed token from server
- `src/pages/Scanner.jsx` → call `/checkin` and `/stats`

## 🛡️ Production Notes
- Never encode raw user info in QR. Use a **signed token** (JWT or HMAC) generated by the backend.
- Validate and rate-limit check‑ins. Keep an audit log.
- Use HTTPS + secure CORS settings.
- For PWA/offline check‑ins, consider background sync for poor connectivity gates.
